Automated IaC Cost Estimation

Calculate infrastructure cost estimates for pull request changes using an automated GitHub Action powered by LLMs.

GitHub Repo Marketplace

Key Features

Automated Analysis

Calculates estimated costs from pull request changes using defined pricing models.

Cost Breakdown

Breaks down costs per resource aligned with current cloud provider pricing.

Terraform & Pulumi

Supports both Terraform and Pulumi configurations.

GitHub Integration

Integrates directly into your GitHub workflows for real-time feedback.

Getting Started

Add the IaC Cost Estimator to your GitHub workflow. Use the YAML configuration below and ensure your repository includes the required secrets and inputs. 

name: IaC Cost Estimation

on:
  pull_request:
    types: [opened, synchronize, reopened]

jobs:
  estimate-cost:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: linfieldlabs/iac-cost-estimator@v1
        with:
          iac-stack: "terraform"  # Required: "terraform" or "pulumi"
          iac-dir: "infrastructure"  # Optional: defaults to repository root
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
          AWS_ACCESS_KEY_ID: ${{ secrets.AWS_ACCESS_KEY_ID }}
          AWS_SECRET_ACCESS_KEY: ${{ secrets.AWS_SECRET_ACCESS_KEY }}

Repository Secrets: OPENAI_API_KEY, AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY, GITHUB_TOKEN
Action Inputs: iac-stack (required) and iac-dir (optional)

Process

Upon a pull request event, the action executes these steps:

  • Checks out the repository and reads the designated IaC files.
  • Analyzes configuration changes to generate an execution plan.
  • Calculates estimated costs using static pricing models.
  • Posts a detailed cost breakdown as a comment on the pull request.

Limitations

Cost estimates are based on predefined pricing models and may not reflect real-time fluctuations, regional pricing differences, or usage variations. The tool currently supports only Terraform and Pulumi; additional platforms may need custom integration.
